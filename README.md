# MyUtils
常用工具类
## 项目中封装了常用的工具类
- ActivityCollector 
主要用于管理项目中的activity，当要退出activity时销毁所有activity
- CameraUtils 
主要用于调用系统相机，进行权限的申请，拍照，存储，剪切；
剪切可以采用系统相机剪切，也集成了Ucrop开源框架进行剪切；
- LitePrefUtils 
封装了sharePrenance的常用操作，一键读取项目存储、读取sp数据；
- retrofit 文件夹
封装了Rxjava + retrofit + 自定义拦截器，方便快捷的retrofit请求框架封装，以扩展，使用简单方便
